Bienvenido administrador

 Esta aplicacion ha sido desarrollada para la Administración Y Gestión de Información en una Clínica Médica orientada al sector Odontológico, con un estilo de Gestion ERP para Control del Modelo De Negocio y funcionalidad integradas para Clientes para Servicio hacia estos mismos, asimismo tendrá una adaptabilidad adicional para los Médicos Usuarios de esta aplicacion, para el control y monitoreo de sus Pacientes


Sus funciones :

  • - Gestionar las categorías(Servicios)
  • - Gestionar los medicos
  • - Distribuir medicos a distintos Servicios/Departamentos
  • - Gestionar las carrito de Compras/Almacen/Stock
